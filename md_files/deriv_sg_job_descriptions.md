# Deriv Solutions Pte Ltd - Job Descriptions & Compensation

## Position 1: Cloud Solutions Engineer

### Role Summary
Design and implement cloud infrastructure solutions for enterprise clients across Singapore and the region.

### Responsibilities
- Architect cloud solutions on AWS, Azure, and Google Cloud Platform
- Lead cloud migration projects for enterprise clients
- Implement DevOps practices and CI/CD pipelines
- Provide technical consultation and solution design
- Collaborate with Malaysia team on cross-border projects
- Mentor junior engineers and conduct knowledge sharing sessions

### Qualifications
- Bachelor's degree in Computer Science, Engineering, or related field
- 3+ years experience in cloud infrastructure and solutions architecture
- Strong proficiency in AWS/Azure/GCP services
- Experience with Infrastructure as Code (Terraform, CloudFormation)
- Certifications preferred: AWS Solutions Architect, Azure Administrator, or equivalent
- Excellent communication skills in English

### Reporting Structure
Reports to Engineering Manager (Singapore)

### Compensation Band
**SGD 6,000 - SGD 9,000/month** (14-month package including AWS)

### Benefits
- CPF contributions (Employer: 17%, Employee: 20% of Ordinary Wages)
- Medical insurance (SGD 80,000 annual limit, outpatient included)
- Dental coverage (SGD 500/year)
- Annual wellness allowance (SGD 500)
- SGD 2,500/year professional development budget
- Annual performance bonus (up to 3 months)
- Stock options after 1 year (for Deriv Group)

### Work Model
Hybrid (2 days office, 3 days remote)

### Career Path
Cloud Engineer → Senior Cloud Engineer → Cloud Architect → Principal Architect

- Develop RESTful APIs using Node.js/Python
- Integrate applications with cloud services (AWS Lambda, Azure Functions)
- Collaborate with UX designers and cloud architects
- Write clean, maintainable code with comprehensive documentation
- Participate in agile development and code reviews

- Bachelor's degree in Computer Science or related field
- 2+ years experience in full stack development

- Good communication skills in English
