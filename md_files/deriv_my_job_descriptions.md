# Deriv Solutions Sdn Bhd - Job Descriptions & Compensation

## Position 1: Full Stack Developer

### Role Summary
Develop and maintain web applications using modern frameworks.

### Responsibilities
- Build responsive front-end interfaces (React, Vue.js)
- Develop RESTful APIs using Node.js/Python
- Collaborate with UX designers and product managers
- Write clean, maintainable code with documentation
- Participate in code reviews and agile ceremonies

### Qualifications
- Bachelor's degree in Computer Science or related field
- 2+ years experience in full stack development
- Proficient in JavaScript, HTML/CSS, and backend frameworks
- Experience with Git, Docker, and cloud platforms (AWS/Azure)
- Good communication skills in English and Bahasa Malaysia

### Reporting Structure
Reports to Engineering Team Lead

### Compensation Band
**RM 5,500 - RM 8,500/month** (based on experience)

### Benefits
- EPF (Employer: 13%, Employee: 11%)
- SOCSO and EIS contributions
- Medical insurance (RM 50,000 annual limit, includes dependents)
- Annual performance bonus (up to 2 months)
- RM 1,500/year learning & development allowance
- Flexible working hours

### Work Model
Hybrid (3 days office, 2 days remote)

### Career Path
Junior Developer → Full Stack Developer → Senior Developer → Tech Lead
